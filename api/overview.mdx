# API Overview

Use the API to integrate Genji platform capabilities into your applications. The platform exposes separate base URLs for production and development to simplify testing and rollout.

## Base URLs
- **Production**: https://api.genji.pk
- **Development**: https://dev-api.genji.pk

## Authentication
- Use bearer tokens to authenticate requests.
- Rotate tokens per environment and keep credentials out of client-side code.

## Next Steps
- Explore the OpenAPI references for request/response shapes.
- Generate SDKs from the schemas and validate against sandbox data before deploying to production.
